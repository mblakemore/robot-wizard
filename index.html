After you download

  1. Unzip into an empty repo: unzip brain.zip -d ./my-agent && cd my-agent
  2. git init && git add -A && git commit -m "C0: bootstrap"
  3. Start an assistant session in the directory: claude (or your CLI of choice)
  4. Paste: @README.md @CLAUDE.md Follow the instructions and begin the loop.

The agent reads CLAUDE.md, initializes state, and runs the first cycle. Each cycle ends with a commit — git log becomes the brain's narrative.
